Для начала не обходимо зайти на github. Потом создать там ветку.

Далее вернуться в локальную папку из которой нужно отправить на git свои данные.

Теперь пишем код:


git init

git add .

git commit -a -m 'mytetra'

git remote add origin git@github.com:KAlex-git/mytetra.git

git branch -M KAlex-git

git push -u origin KAlex-git
