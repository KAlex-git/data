Чтобы пользовательский cron работал и запустился, необходимо:


sudo crontab -u $USER -e


$USER - это имя текущего пользователя


Как редактировать пользовательский crontab -l через СВОЙ редактор, например, VIM?


1. Создать файл с именем $USER

2. Изменить:



привелегии пользователя

sudo chmod 600

права

sudo chown -R ${USER}:${GROUP}


3. Файл должен находиться в каталоге: /var/spool/cron/


Вот, наглядный пример:


.rw------- 834 kalex 17 янв 10:37  /var/spool/cron/kalex


Syntax Crontab
